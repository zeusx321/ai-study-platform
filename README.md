# 🎓 AI Study Platform

An AI-powered educational workspace that helps students organize their study materials, interact with AI assistants, generate quizzes, and receive context-aware explanations using Agentic AI and Retrieval-Augmented Generation (RAG).

---

## 🚀 Features

### 📚 Study Workspace
- Organize subjects and study materials
- Create and manage notes
- Upload learning resources
- Build a personal learning environment

### 🤖 AI Assistant
- Ask questions about uploaded documents
- Generate summaries
- Create adaptive quizzes
- Get step-by-step explanations
- Receive personalized study guidance

### 📄 Smart Document Processing
- Support for PDF, DOCX, and TXT files
- Automatic text extraction
- Semantic search with vector embeddings
- Context-aware responses using RAG

### ♿ Accessibility
- Keyboard navigation
- Screen reader support
- RTL compatibility
- Responsive design
- Simplified learning mode
